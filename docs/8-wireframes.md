## SCR-01 로그인 화면

> 참조 기능 ID: F-A-01, F-A-02, F-A-03

### 모바일 (360px)

```
+----------------------------------+
|          todolist-app            |
|                                  |
|    +----------------------------+|
|    |        로그인              ||
|    +----------------------------+|
|                                  |
|    이메일                        |
|    +----------------------------+|
|    | user@example.com           ||
|    +----------------------------+|
|                                  |
|    비밀번호                      |
|    +----------------------------+|
|    | ••••••••                   ||
|    +----------------------------+|
|                                  |
|    +----------------------------+|
|    |  [ ! ] 이메일 또는 비밀번호 ||
|    |        가 올바르지 않습니다.||
|    +----------------------------+|
|    (로그인 실패 시에만 표시)      |
|                                  |
|    +----------------------------+|
|    |  [ ! ] 로그인 5회 실패로   ||
|    |  계정이 잠겼습니다.         ||
|    |  관리자에게 문의하세요.     ||
|    +----------------------------+|
|    (계정 잠금 시에만 표시)        |
|                                  |
|    +----------------------------+|
|    |         로그인             ||
|    +----------------------------+|
|                                  |
|    계정이 없으신가요?             |
|         [ 회원가입 ]             |
|                                  |
+----------------------------------+
```

**인터랙션 설명 (모바일)**

- 이메일/비밀번호 입력 후 로그인 버튼 탭 시 F-A-01 인증 요청
- 인증 실패 시 에러 메시지 영역이 인라인으로 나타남 (기본 상태에서는 숨김)
- 5회 연속 실패 시 에러 메시지가 잠금 안내 메시지로 교체되고 로그인 버튼 비활성화 (F-A-03)
- "회원가입" 링크 탭 시 SCR-02로 이동 (F-A-02)
- 비밀번호 필드는 입력 내용을 마스킹 처리

## SCR-02 회원가입 화면

> 참조 기능 ID: F-A-04, F-A-05, F-A-06

### 모바일 (360px)

```
+----------------------------------+
|          todolist-app            |
|                                  |
|    +----------------------------+|
|    |        회원가입            ||
|    +----------------------------+|
|                                  |
|    이메일                        |
|    +----------------------------+|
|    | user@example.com           ||
|    +----------------------------+|
|    [ ! ] 이미 사용 중인 이메일   |
|           입니다. (에러 시)       |
|                                  |
|    비밀번호                      |
|    +----------------------------+|
|    | ••••••••                   ||
|    +----------------------------+|
|                                  |
|    비밀번호 정책                 |
|    +----------------------------+|
|    | [v] 8~20자                 ||
|    | [ ] 영문 대문자 1자 이상   ||
|    | [v] 영문 소문자 1자 이상   ||
|    | [ ] 숫자 1자 이상          ||
|    | [ ] 특수문자(!@#$%^&*) 1자 ||
|    +----------------------------+|
|                                  |
|    비밀번호 확인                 |
|    +----------------------------+|
|    | ••••••••                   ||
|    +----------------------------+|
|    [ ! ] 비밀번호가 일치하지     |
|           않습니다. (에러 시)     |
|                                  |
|    +----------------------------+|
|    |         회원가입           ||
|    +----------------------------+|
|                                  |
|    이미 계정이 있으신가요?        |
|         [ 로그인 ]               |
|                                  |
+----------------------------------+
```

**인터랙션 설명 (모바일)**

- 비밀번호 입력 시 정책 체크리스트가 실시간으로 업데이트 (충족 항목: [v], 미충족: [ ])
- 비밀번호 확인 필드 blur 시 일치 여부 검증, 불일치 시 인라인 에러 표시
- 모든 정책 미충족 또는 이메일 형식 오류 시 회원가입 버튼 비활성화
- 회원가입 성공 시 이메일 인증 안내 화면으로 전환 (F-A-05)
- "로그인" 링크 탭 시 SCR-01로 이동 (F-A-06)

```

## SCR-03 메인 목록 화면

> 참조 기능 ID: F-T-01, F-T-02, F-T-03, F-T-04, F-T-05, F-T-06, F-T-07, F-T-08

### 모바일 (360px)

```
+----------------------------------+
| todolist-app   [통계] [로그아웃] |
+----------------------------------+
|                                  |
|  +-----+ +-----+ +-----+ +----+ |
|  | 전체| |시작 | |진행 | |종료| |
|  +-----+ | 전  | | 중  | |됨  | |
|           +-----+ +-----+ +----+ |
|                                  |
|  +----------+ +----------+      |
|  |기간 내완료| |기간초과완료|    |
|  +----------+ +----------+      |
|                                  |
|  정렬 기준: [ 종료일     v ]    |
|  정렬 방향: [ 오름차순   v ]    |
|                                  |
+----------------------------------+
| [ ] 프로젝트 기획서 작성         |
|     2026-03-01 ~ 2026-04-30      |
|     [진행 중]                    |
+----------------------------------+
| [v] 디자인 시스템 구축           |
|     2026-02-01 ~ 2026-03-31      |
|     [기간 내 완료]               |
+----------------------------------+
| [ ] 백엔드 API 개발              |
|     2026-04-01 ~ 2026-05-15      |
|     [시작 전]                    |
+----------------------------------+
| [ ] 레거시 코드 리팩토링         |
|     2026-01-01 ~ 2026-02-28      |
|     [종료됨]                     |
+----------------------------------+
| [v] 배포 환경 설정               |
|     2026-03-01 ~ 2026-04-10      |
|     [기간 초과 완료]             |
+----------------------------------+
|                                  |
|    < 1  2  3  4  5  6  7 >      |
|                                  |
+----------------------------------+
|                                  |
|           [+ 할일 추가]          |
|                                  |
+----------------------------------+
```

**빈 목록 EmptyState (모바일)**

```
+----------------------------------+
| todolist-app   [통계] [로그아웃] |
+----------------------------------+
|  [ 전체 ] [시작전] [진행중] ... |
+----------------------------------+
|                                  |
|                                  |
|       ( 할일 없음 아이콘 )       |
|                                  |
|    아직 등록된 할일이 없습니다.  |
|    첫 번째 할일을 추가해보세요!  |
|                                  |
|    +----------------------------+|
|    |       + 할일 추가          ||
|    +----------------------------+|
|                                  |
+----------------------------------+
```

**인터랙션 설명 (모바일)**

- 상태 필터 탭은 가로 스크롤 가능 (overflow-x: auto), 현재 선택 탭 강조
- 정렬 기준/방향은 드롭다운 셀렉트 박스 (네이티브 select)
- 목록 행 탭 시 SCR-05 상세 화면으로 이동 (F-T-06)
- 체크박스 탭 시 즉시 완료 처리/해제, 완료 해제 시 확인 메시지 표시 (F-T-07)
- [+ 할일 추가] 버튼 탭 시 SCR-04로 이동 (F-T-05)
- 빈 목록 시 EmptyState 표시 (할일 추가 버튼 포함)
- 페이지네이션 20건 단위, 현재 페이지 강조 (F-T-04)

## SCR-04 할일 등록 화면

> 참조 기능 ID: F-T-09

### 모바일 (360px)

```
+----------------------------------+
| < 목록으로                       |
+----------------------------------+
|                                  |
|    +----------------------------+|
|    |        할일 등록           ||
|    +----------------------------+|
|                                  |
|    제목 *                        |
|    +----------------------------+|
|    | 할일 제목을 입력하세요      ||
|    +----------------------------+|
|                            0/200 |
|    [ ! ] 제목은 필수 항목입니다  |
|           (에러 시)              |
|                                  |
|    설명                          |
|    +----------------------------+|
|    |                            ||
|    | 할일에 대한 설명을 입력    ||
|    | 하세요. (선택 사항)        ||
|    |                            ||
|    +----------------------------+|

- 완료율, 기간 내 완료율의 소수점은 1자리까지 표시
- 데이터가 0인 경우: 건수 카드 "0건" 표시, progress bar 0%, 기간 초과 건수 "0건" 표시

## 공통 컴포넌트 정의

### 상태 뱃지 (Status Badge)

```
+------------------+
| [시작 전]         |   색상: 회색  (UPCOMING)
| [진행 중]         |   색상: 파랑  (IN_PROGRESS)
| [종료됨]          |   색상: 빨강  (OVERDUE)
| [기간 내 완료]    |   색상: 초록  (COMPLETED_ON_TIME)
| [기간 초과 완료]  |   색상: 주황  (COMPLETED_LATE)
+------------------+
```

### 확인 다이얼로그 (Confirm Dialog)

```
+------------------------------------+
|  (dimmed overlay)                  |
|                                    |
|  +--------------------------------+|
|  |  다이얼로그 제목               ||
|  |                                ||
|  |  본문 메시지 (1~3줄)           ||
|  |  [ ! ] 경고 메시지 (조건부)    ||
|  |                                ||
|  |   +-----------+ +------------+||
|  |   |   취  소  | |   확  인   |||
|  |   +-----------+ +------------+||
|  +--------------------------------+|
|                                    |
+------------------------------------+
```

### 토스트 메시지 (Toast)

```
모바일: 화면 하단 중앙 고정
데스크탑: 화면 우상단 고정

+----------------------------------+
|  [ v ] 저장되었습니다.            |  (성공: 초록)
+----------------------------------+
+----------------------------------+
|  [ ! ] 오류가 발생했습니다.       |  (실패: 빨강)
+----------------------------------+
```

### 헤더 (Header)

```
모바일:
+----------------------------------+
| todolist-app   [통계] [로그아웃] |
+----------------------------------+

데스크탑:
+------------------------------------------------------------------------+
|  todolist-app                          [ 통계 ]       [ 로그아웃 ]    |
+------------------------------------------------------------------------+
```
