# 문제 정의 :                                                                   
- 할일 관리 앱은 나만의 할일 관리가 힘들다.                                     
- 할일 관리 앱에 해당 할일의 종료일을 지정하는 애플리케이션이 없다.             
- 사용자 인증 후에 나만의 할일 관리 기능이 필요하다.                            
                                                                                
# todolist-app 애플리케이션의 주요 기능 :                                       
- 모든 사용자는 회원 가입 후에 로그인하여 할일 관리 기능을 이용한다.            
- 모든 개인별 할일 데이터에는 할일의 시작일과 종료일 이 지정되어야 한다.        
- 할일은 진행중인 할일, 종료된 할일, 시작전인 할일, 성공적으로 완료된 할일,     
완료하지 못한 할일을 필터링하여 볼 수 있도록 한다.                              
- 할일은 정렬 기준을 지정해 몇가지 기준(종료일, 시작일 등)을 이용해 정렬할 수   
있어야 한다.                                                                    
- 할일은 완료하면 완료 여부를 업데이트 할 수 있도록 한다.                       
- 할일 통계 정보도 보여줘야해

   ID 참조가 없음
